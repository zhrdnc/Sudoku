# Sudoku CLI Game (C)
A lightweight Sudoku engine focusing on core programming logic and matrix manipulations.

Highlights
-Logic: Validation of rows, columns, and 3x3 grids.

-I/O: Ability to load and save puzzles via .txt files.

-Reliability: Built-in fallback matrix for missing files.

-UX: Real-time feedback for user moves and game status.
